# Express-empty
Empty js express project



TO INSTALL DEPENDENCES
<code>
	npm install 
</code>

TO START THE SERVER
Linux:
<code>
	DEBUG=myapp ./bin/www
</code>
	Windows: 
<code>
	set DEBUG=myapp & npm start
</code>
